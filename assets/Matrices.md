# Matrices
==========

## Definition
A matrix is a rectangular array of numbers.

## Types of Matrices
* **Square Matrix**: Same number of rows and columns.
* **Identity Matrix**: Diagonal elements are 1.

## Matrix Operations
* **Addition**: Matrix sum.
* **Multiplication**: Matrix product.
