# Determinants
=============

## Definition
A determinant is a scalar value.

## Properties
* **Zero Determinant**: Singular matrix.
* **Non-Zero Determinant**: Invertible matrix.

## Calculating Determinants
* **Cofactor Expansion**: Recursive formula.
* **LU Decomposition**: Factorization.

## Example
Matrix A = [[1, 2], [3, 4]]

det(A) = 1 × 4 - 2 × 3 = -2
